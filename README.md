# Текстовий редактор

Сучасний текстовий редактор з функціями обробки тексту, історією змін та іншими корисними можливостями.

## Використані технології

- **React** - JavaScript-бібліотека для створення інтерфейсів користувача
- **Vite** - інструмент збірки, що забезпечує швидкий запуск та гарячу перезбірку
- **Tailwind CSS** - фреймворк для стилізації з попередньо визначеними класами
- **JavaScript (ES6+)** - мова програмування для реалізації логіки застосунку
- **HTML5 та CSS3** - для структури сторінки та стилів
- **Local Storage API** - для збереження тексту в браузері
- **Clipboard API** - для копіювання тексту в буфер обміну
- **FileReader API** - для завантаження текстових файлів
- **Blob API** - для завантаки тексту як файлу

## Як запустити локально

1. Клонуйте репозиторій:

   ```bash
   git clone <your-repository-url>
   ```

2. Встановіть залежності:

   ```bash
   cd text-editor
   npm install
   ```

3. Запустіть проект у режимі розробки:

   ```bash
   npm run dev
   ```

4. Відкрийте браузер за адресою, що вказана в терміналі (зазвичай http://localhost:5173)

Щоб зібрати проект для продакшну:

```bash
npm run build
```

## Рішення щодо продуктивності та механізму undo/redo

### Продуктивність:

- Використовується оптимізований механізм обробки текстових операцій, який працює з рядками ефективно
- Для кожної операції вимірюється час виконання за допомогою `performance.now()`, що дозволяє відстежувати продуктивність
- Використовується локальне сховище для автозбереження тексту з затримкою 1 секунда, щоб уникнути надмірного запису
- Всі операції обробляються асинхронно, щоб не блокувати інтерфейс користувача
- Використовується `useMemo` для оптимізації обчислення метрик тексту (кількість слів, символів, рядків)
- Використовується `useCallback` для мемоїзації функцій, що передаються в компоненти

### Механізм undo/redo:

- Реалізовано через кастомний хук `useHistory`, який зберігає історію змін тексту
- Обмежена кількість збережених станів (за замовчуванням 10) для економії пам'яті
- При кожному змінюванні тексту новий стан додається до історії
- Підтримуються стандартні функції скасування (undo) та повторення (redo) з перевіркою можливості виконання
- Індекс поточного стану в історії відстежується окремо для правильного перемикання між станами

## Функціональні можливості

- Редагування тексту з візуальними підказками
- Перетворення регістру (верхній, нижній, перша літера велика)
- Очищення тексту (видалення пробілів, табів, спецсимволів)
- Форматування тексту (додавання лапок, дужок, тире)
- Пошук та заміна тексту
- Сортування рядків (за алфавітом, в зворотньому порядку)
- Видалення дублікатів
- Завантаження та збереження файлів
- Копіювання в буфер обміну
- Підрахунок кількості слів, символів та рядків
